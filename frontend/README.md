# Frontend

The frontend is a React + Vite client for EnglishBuddy.

## Features

- Registration and login flows
- Protected dashboard
- Practice session composer
- Weekly progress cards and recent history

## Commands

- `npm run dev` to start the frontend
- `npm run build` to create a production build
- `npm run lint` to run ESLint

## API configuration

The client points to `http://localhost:8080/api` by default. Override it with
`VITE_API_URL` if your backend runs elsewhere.
